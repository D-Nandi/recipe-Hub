# Flavour-Hub
Flavour Hub is a modern recipe discovery platform that helps users explore delicious recipes with smart search, multilingual support, and a cinematic user experience.


🍽️ Features
🔍 Smart recipe search
🌍 Multi-language support
⚡ Fast and optimized translation engine
🧠 Intelligent caching system
📱 Responsive modern UI
🎨 Cinematic dark design
🍲 Recipe discovery experience
🚀 Performance-optimized frontend


🌐 Multilingual Translation System
Recipe Hub includes a custom translation engine with:
Progressive translation loading
Persistent local cache
Smart text filtering
Parallel translation batching
Optimized DOM translation
Multi-language UI rendering

Supported Languages:
English
हिन्दी (Hindi)
বাংলা (Bengali)


🎨 UI & Experience
The platform is designed with:
Modern glassmorphism aesthetics
Smooth animations
Cinematic dark ambience
Responsive layouts
Clean user interactions


🛠️ Tech Stack
HTML5
CSS3
Vanilla JavaScript
Translation APIs
LocalStorage caching
Custom frontend architecture
